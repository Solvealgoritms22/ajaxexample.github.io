AJAX: Asynchronous JavaScript and XML

¿Qué es AJAX?

AJAX, que significa "Asynchronous JavaScript and XML", es una técnica de desarrollo web para crear aplicaciones interactivas. Permite actualizar partes del contenido de una página web sin recargar toda la página.

Ejemplo de Funcionamiento de AJAX

Imagina una aplicación de correo donde puedes marcar mensajes como leídos sin recargar la página. Esto se logra mediante una solicitud AJAX que envía la información al servidor y actualiza la interfaz de usuario en base a la respuesta del servidor.

Código de Ejemplo

// Crear un nuevo objeto XMLHttpRequest
let xhr = new XMLHttpRequest();

// Configurar la solicitud
xhr.open('GET', 'server-url', true);

// Respuesta del servidor
xhr.onreadystatechange = function() {
    if (xhr.readyState == 4 && xhr.status == 200) {
       console.log(xhr.responseText);
    }
};

// Enviar la solicitud
xhr.send();
